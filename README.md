# RAPIFIX 🚀

RAPIFIX is a MERN-stack doorstep appliance repair platform.

## Features

- User Authentication
- Service Booking
- Technician Registration
- Admin Dashboard

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Installation

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
